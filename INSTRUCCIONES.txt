Para levantar el backend (Python 3.6 + Fastapi + uvicorn + SqlAlchemy) puede correr el archivo python en el terminal python con:
python main.py

Para levantar el frontend (nodejs + reactJS) es necesario poner los comandos:
npm install 
npm start

Posdata:
Se escogio como base de datos Sqlite, y se encuentra en la carpeta backend

localhost:4200/
